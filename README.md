# 📧 Email Spam Detection using Machine Learning

## 📌 Project Overview

This project builds a Machine Learning model to classify messages/emails as **Spam** or **Not Spam (Ham)**.
It uses **TF-IDF text vectorization** and a **Logistic Regression classifier** trained on labeled SMS/email data.

The system allows users to input a message and instantly receive a spam prediction.

---

## 🚀 Features

* Text preprocessing and cleaning
* TF-IDF feature extraction
* Logistic Regression spam classifier
* Model & vectorizer saved as `.pkl` files
* Command-line prediction tool (`predict.py`)
* Beginner-friendly project structure

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
